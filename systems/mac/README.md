# Mac

## Setup

* Do a system update
* Install XCode
* Install [Homebrew](https://brew.sh/)
* Run the initial setup through the terminal
* Set up programs and system preferences

### Terminal

```sh
mkdir -p code/dguo
cd code/dguo
git clone https://github.com/dguo/dotfiles.git
cd dotfiles
./configure.sh
```
